  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install Python packages:
      
      ```bash
      git clone https://github.com/Techdude01/NBAPredict.git
      cd NBAPredict
      pip install -r requirements.txt
      ```
      
      The project requires Python 3.10+ and packages including pandas, numpy, tensorflow, scikit-learn, and kaggle.
    </Step>

    <Step title="Set up Kaggle API credentials">
      Create a `.env` file with your Kaggle credentials:
      
      ```bash
      cp .env.example .env
      ```
      
      Edit `.env` and add your Kaggle username and API key:
      
      ```bash
      KAGGLE_USERNAME=your_username
      KAGGLE_KEY=your_api_key
      ```
      
      <Tip>Get your Kaggle API credentials from your [Kaggle account settings](https://www.kaggle.com/settings)</Tip>
    </Step>

    <Step title="Run data preparation">
      Open and execute `NBAPredict.ipynb` to download the NBA dataset and prepare training data:
      
      ```python
      # The notebook will:
      # 1. Download historical NBA game data from Kaggle
      # 2. Aggregate per-team-season statistics
      # 3. Create train/validation/test splits
      # 4. Generate scaled and polynomial features
      ```
      
      This creates processed data files in `./data/processed/` including:
      - `X_train_scaled.npy`, `X_val_scaled.npy`, `X_test_scaled.npy`
      - `X_train_poly2.npy`, `X_train_poly3.npy`, `X_train_pca.npy`
      - `y_train.npy`, `y_val.npy`, `y_test.npy`
    </Step>

    <Step title="Train models">
      Run any of the model notebooks to train and evaluate:
      
      - **LogPredict.ipynb** — Custom logistic regression with gradient ascent
      - **KNNPredict.ipynb** — KNN with grid search over K, distance metric, and weighting
      - **NeuralNetworkPredict.ipynb** — TensorFlow/Keras neural networks with various architectures
      
      Each notebook trains multiple configurations and saves the best model to `./results/`.
    </Step>

    <Step title="Compare results">
      Run `ModelComparison.ipynb` to load the best models and compare performance:
      
      ```python
      # Metrics evaluated:
      # - Accuracy
      # - Precision
      # - Recall
      # - F1 Score
      # - AUC (Area Under ROC Curve)
      ```
      
      The comparison notebook generates visualizations and summary tables showing which model performs best for playoff prediction.
    </Step>
  </Steps>
