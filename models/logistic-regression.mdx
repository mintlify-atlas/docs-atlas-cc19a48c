# Logistic Regression Model

## Overview

The logistic regression model uses a custom gradient ascent implementation with L2 regularization to predict NBA playoff team qualification. This classical approach provides a strong baseline with interpretable results.

## Model Architecture

### Core Components

<Steps>
  <Step title="Sigmoid Function">
    Transforms linear predictions to probabilities:
    ```python
    def sigmoid(z):
        return 1 / (1 + np.exp(-z))
    ```
  </Step>
  
  <Step title="Hypothesis Function">
    Combines features with learned weights:
    ```python
    def hypothesis(X, w):
        return sigmoid(np.dot(X, w))
    ```
  </Step>
  
  <Step title="Log-Likelihood with Regularization">
    Objective function with L2 penalty:
    ```python
    def log_likelihood(X, y, w, lambda_reg=0):
        hyp = hypothesis(X, w)
        log_lik = np.sum(y * np.log(hyp + 1e-10) + 
                        (1 - y) * np.log(1 - hyp + 1e-10))
        if lambda_reg > 0:
            reg_term = -lambda_reg * np.sum(w[1:] ** 2) / 2
            log_lik += reg_term
        return log_lik
    ```
  </Step>
</Steps>

## Gradient Ascent Implementation

### Training Procedure

The model uses gradient ascent to maximize the log-likelihood:

```python
def Gradient_Ascent(X, y, learning_rate, num_iters, lambda_reg=0):
    w = np.zeros((X.shape[1], 1))
    
    for i in range(num_iters):
        hyp = hypothesis(X, w)
        gradient = np.dot(X.T, (y - hyp)) / N
        
        # Add L2 regularization gradient
        if lambda_reg > 0:
            w_without_bias = w.copy()
            w_without_bias[0] = 0
            gradient -= lambda_reg * w_without_bias
        
        w = w + learning_rate * gradient
    
    return w
```

## L2 Regularization

### Purpose

L2 regularization (Ridge regression) adds a penalty term to the log-likelihood:

- **Prevents overfitting** by constraining weight magnitudes
- **Improves generalization** to unseen data
- **Does NOT penalize** the bias term (intercept)

### Effect on Model

The regularization term modifies the gradient:

```python
gradient -= lambda_reg * w_without_bias
```

This shrinks weights toward zero, creating smoother decision boundaries.

## Predictions

### Classification Function

```python
def predict_class(X, w, threshold=0.5):
    y_prob = hypothesis(X, w)
    y_pred = (y_prob >= threshold).astype(int)
    return y_pred
```

### Probability Outputs

The model outputs probabilities that a team qualifies for the playoffs:

- **> 0.5:** Predicted playoff team
- **≤ 0.5:** Predicted non-playoff team
