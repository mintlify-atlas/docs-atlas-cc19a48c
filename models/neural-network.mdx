# Neural Network Model

### Regularization

<AccordionGroup>
  <Accordion title="L2 Regularization (Ridge)">
    Adds penalty term to loss function:
    ```python
    loss = binary_crossentropy + lambda * sum(w^2)
    ```
    
    **Tested λ values:** 0, 0.001, 0.01, 0.1, 1, 10
    
    **Best λ:** 0.01
    
    Prevents overfitting by penalizing large weights.
  </Accordion>
  
  <Accordion title="Early Stopping">
    ```python
    EarlyStopping(
        monitor='val_loss',
        patience=20,
        restore_best_weights=True
    )
    ```
    
    Stops training when validation loss stops improving for 20 consecutive epochs.
  </Accordion>

## Implementation

### Model Definition

```python
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers, models
from tensorflow.keras.regularizers import l2

def build_model(input_dim, layer_sizes, l2_reg):
    model = models.Sequential([
        layers.Input(shape=(input_dim,)),
    ])
    
    # Add hidden layers with ReLU activation
    for units in layer_sizes:
        model.add(layers.Dense(
            units, 
            activation='relu',
            kernel_regularizer=l2(l2_reg) if l2_reg > 0 else None
        ))
    
    # Output layer with sigmoid activation
    model.add(layers.Dense(1, activation='sigmoid'))
    
    # Compile model
    model.compile(
        optimizer=keras.optimizers.Adam(learning_rate=0.001),
        loss='binary_crossentropy',
        metrics=['accuracy']
    )
    
    return model
```

### Training the Model

```python
from tensorflow.keras.callbacks import EarlyStopping

# Best hyperparameters
layer_sizes = [128, 64]
l2_reg = 0.01

# Build model
model = build_model(
    input_dim=X_train.shape[1],
    layer_sizes=layer_sizes,
    l2_reg=l2_reg
)

# Early stopping callback
early_stop = EarlyStopping(
    monitor='val_loss',
    patience=20,
    restore_best_weights=True,
    verbose=0
)

# Class weights for imbalanced data
class_weight = {
    0: len(y_train) / (2 * (len(y_train) - y_train.sum())),
    1: len(y_train) / (2 * y_train.sum())
}

# Train model
history = model.fit(
    X_train, y_train,
    validation_data=(X_val, y_val),
    epochs=200,
    batch_size=32,
    class_weight=class_weight,
    callbacks=[early_stop],
    verbose=0
)

# Evaluate on test set
y_test_pred_prob = model.predict(X_test).flatten()
y_test_pred = (y_test_pred_prob > 0.5).astype(int)

from sklearn.metrics import accuracy_score, roc_auc_score
print(f"Test Accuracy: {accuracy_score(y_test, y_test_pred):.4f}")
print(f"Test AUC: {roc_auc_score(y_test, y_test_pred_prob):.4f}")
```
