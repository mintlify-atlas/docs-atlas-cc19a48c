# K-Nearest Neighbors (KNN) Model

## Overview

The K-Nearest Neighbors classifier predicts playoff qualification based on the similarity of team statistics to historical playoff and non-playoff teams. This non-parametric approach makes no assumptions about the underlying data distribution.

## Model Concept

<Steps>
  <Step title="Distance Calculation">
    Calculate the distance between the test team and all training teams using a distance metric (Euclidean or Manhattan).
  </Step>
  
  <Step title="Find Nearest Neighbors">
    Identify the K closest training examples based on calculated distances.
  </Step>
  
  <Step title="Majority Vote">
    Predict the class based on the majority label among the K nearest neighbors (with optional distance weighting).
  </Step>
</Steps>

## Hyperparameters Tested

### K Values (Number of Neighbors)

<AccordionGroup>
  <Accordion title="K = 1">
    **Use Case:** Highly flexible, captures local patterns

  <Accordion title="K = 3-11">
    **Use Case:** Balanced flexibility and smoothness

  </Accordion>
  
  <Accordion title="K = 15-21">
    **Use Case:** Smoother decision boundaries
    
    **Risk:** May oversmooth important patterns
    
    **Best K:** **K=21** with Manhattan distance and Polynomial Degree 3 features
  </Accordion>
</AccordionGroup>

### Distance Metrics

<CodeGroup>
```python Euclidean Distance
# L2 norm - straight-line distance
dist = sqrt(sum((x_i - y_i)^2))

# Good for: Continuous features, isotropic data
```

```python Manhattan Distance
# L1 norm - city-block distance
dist = sum(|x_i - y_i|)

# Good for: High-dimensional data, grid-like patterns
# BEST PERFORMER for this dataset
```
</CodeGroup>

### Weighting Schemes

<Tabs>
  <Tab title="Uniform">
    All K neighbors have equal voting power:
    ```python
    prediction = mode(labels_of_k_neighbors)
    ```
  </Tab>
  
  <Tab title="Distance-Weighted">
    Closer neighbors have more influence:
    ```python
    weights = 1 / (distances + epsilon)
    prediction = weighted_vote(labels, weights)
    ```
    
    <Note>
    Distance weighting achieved the **best performance** for the optimal model.
    </Note>
  </Tab>
</Tabs>

## Model Strengths

<AccordionGroup>
  <Accordion title="No Training Phase">
    KNN is a lazy learner - no explicit training step required. Predictions are made by directly comparing to stored examples.
  </Accordion>
  
  <Accordion title="Non-Parametric">
    Makes no assumptions about data distribution, allowing it to capture complex patterns.
  </Accordion>
  
  <Accordion title="Naturally Multi-Class">
    Can easily handle classification problems with more than two classes without modification.
  </Accordion>
  
  <Accordion title="High Recall">
    Achieved **98.44% recall** - excellent at identifying all playoff teams.
  </Accordion>
  
  <Accordion title="Interpretable Predictions">
    Predictions can be explained by examining the nearest neighbor examples.
  </Accordion>
</AccordionGroup>

## Model Limitations

<Warning>
- **Computationally expensive** - must calculate distances to all training examples for each prediction
- **Memory intensive** - stores entire training dataset
- **Sensitive to feature scaling** - requires normalized features
- **Curse of dimensionality** - performance degrades with very high-dimensional data
- **Lower precision** - 75% precision means some false positives
</Warning>

## Implementation Example

### Training and Prediction

```python
from sklearn.neighbors import KNeighborsClassifier

# Best hyperparameters
k = 21
metric = 'manhattan'
weights = 'distance'

# Load polynomial degree 3 features
X_train_poly3 = np.load('data/processed/X_train_poly3.npy')
X_test_poly3 = np.load('data/processed/X_test_poly3.npy')
y_train = np.load('data/processed/y_train.npy')
y_test = np.load('data/processed/y_test.npy')

# Initialize and train model
knn = KNeighborsClassifier(
    n_neighbors=k,
    metric=metric,
    weights=weights
)
knn.fit(X_train_poly3, y_train)

# Make predictions
y_pred = knn.predict(X_test_poly3)
y_pred_proba = knn.predict_proba(X_test_poly3)[:, 1]

# Evaluate
from sklearn.metrics import accuracy_score, roc_auc_score
print(f"Accuracy: {accuracy_score(y_test, y_pred):.4f}")
print(f"AUC: {roc_auc_score(y_test, y_pred_proba):.4f}")
```

### Hyperparameter Tuning

```python
from sklearn.model_selection import GridSearchCV

# Define parameter grid
param_grid = {
    'n_neighbors': [1, 3, 5, 7, 9, 11, 15, 21],
    'metric': ['euclidean', 'manhattan'],
    'weights': ['uniform', 'distance']
}

# Grid search with cross-validation
knn = KNeighborsClassifier()
grid_search = GridSearchCV(
    knn, param_grid,
    cv=5, scoring='roc_auc',
    n_jobs=-1
)
grid_search.fit(X_train_poly3, y_train)

print(f"Best parameters: {grid_search.best_params_}")
print(f"Best CV AUC: {grid_search.best_score_:.4f}")
```
