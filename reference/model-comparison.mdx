The `ModelComparison.ipynb` notebook loads the best models from each approach and compares their performance across five key metrics on the test set. It provides visualizations, statistical analysis, and recommendations for model selection.

## Purpose

This notebook answers the question: **Which model is best for predicting NBA playoff teams?**

It evaluates:
- Logistic regression (custom implementation)
- K-nearest neighbors (scikit-learn)
- Neural network (TensorFlow/Keras)

Across five metrics:
- Accuracy
- Precision
- Recall
- F1 Score
- AUC (Area Under ROC Curve)

## Prerequisites

You must run all three model training notebooks first:
1. [LogPredict.ipynb](/reference/log-predict)

## Performance comparison

### Overall results

| Model | Accuracy | Precision | Recall | F1 | AUC | Avg Score |
|-------|----------|-----------|--------|-----|-----|------------|
| **Neural Network** | 87.50% | **89.36%** | 98.44% | 93.62% | **98.24%** | **0.9258** |
| **Logistic Regression** | **89.17%** | 84.93% | 96.88% | **90.51%** | 97.54% | **0.9181** |
| **K-Nearest Neighbors** | 81.67% | 75.00% | **98.44%** | 85.14% | 96.60% | **0.8737** |

<Note>
  **Neural Network wins overall** with the highest average score (0.9258) and best AUC (98.24%). Logistic Regression achieves the highest F1 score (90.51%). KNN has the highest recall (98.44%) but lowest precision (75.00%).
</Note>

### Metric-by-metric analysis

<Tabs>
  <Tab title="Accuracy">
    **Winner: Logistic Regression (89.17%)**
    
    Logistic regression correctly classifies the highest percentage of teams. Neural networks are close behind (87.50%).
    
    KNN's lower accuracy (81.67%) reflects its high false positive rate—it over-predicts playoff teams.
  </Tab>
  
  <Tab title="Precision">
    **Winner: Neural Network (89.36%)**
    
    When the neural network predicts "playoff," it's correct 89.36% of the time. Logistic regression is close (84.93%).
    
    KNN has the lowest precision (75.00%)—it flags many non-playoff teams as playoff-bound.
  </Tab>
  
  <Tab title="Recall">
    **Winner: KNN and Neural Network (98.44%)**
    
    Both models successfully identify 98.44% of actual playoff teams. Logistic regression is slightly lower (96.88%).
    
    All three models prioritize recall—missing a playoff team is worse than over-predicting.
  </Tab>
  
  <Tab title="F1 Score">
    **Winner: Neural Network (93.62%)**
    
    The neural network achieves the best balance between precision and recall. Logistic regression is second (90.51%).
    
    KNN's F1 (85.14%) suffers from its low precision despite high recall.
  </Tab>
  
  <Tab title="AUC">
    **Winner: Neural Network (98.24%)**
    
    The neural network's ROC curve dominates—it maintains high true positive rate across all false positive rates.
    
    All three models achieve excellent AUC (>96%), indicating strong separation between playoff and non-playoff teams.
  </Tab>
</Tabs>

## ROC curves

The notebook generates overlaid ROC curves showing true positive rate vs false positive rate at various decision thresholds.

**Key observations**:
- Neural network curve hugs the top-left corner most closely (AUC: 0.9824)
- Logistic regression is very close (AUC: 0.9754)
- KNN follows slightly behind (AUC: 0.9660)
- All three curves are well above the diagonal (random classifier)

## Feature transformation analysis

The notebook also compares how each model performs across different feature transformations:

### Logistic Regression by feature type

| Transformation | Test AUC |
|----------------|----------|
| **Baseline** | **97.54%** |
| Poly2 | 96.88% |
| Poly3 | 95.12% |
| PCA | 96.24% |

Logistic regression works best with baseline features. Polynomial features add unnecessary complexity.

### KNN by feature type

| Transformation | Test AUC |
|----------------|----------|
| Baseline | 94.12% |
| Poly2 | 95.48% |
| **Poly3** | **96.60%** |
| PCA | 92.85% |

KNN benefits from polynomial features—the explicit interactions help identify similar teams.

### Neural Network by feature type

| Transformation | Test AUC |
|----------------|----------|
| **Baseline** | **98.24%** |
| Poly2 | 96.78% |
| Poly3 | 95.42% |
| PCA | 97.15% |

Neural networks perform best with baseline features—they learn interactions internally through hidden layers.

## Model selection guide

<CardGroup cols={3}>
  <Card title="Choose Neural Network if..." icon="brain">
    - You want the highest overall performance
    - You have GPU resources for training
    - Interpretability is not critical
    - You're willing to tune hyperparameters
  </Card>
  
  <Card title="Choose Logistic Regression if..." icon="chart-line">
    - You need interpretable feature weights
    - You want the best F1 score
    - Training speed matters
    - You need to explain predictions to stakeholders
  </Card>
  
  <Card title="Choose KNN if..." icon="location-dot">
    - You need maximum recall (catch all playoff teams)
    - False positives are acceptable
    - You don't need fast predictions
    - You have sufficient memory for training data
  </Card>
</CardGroup>

| Model | Training Time | Prediction Time | Memory Usage |
|-------|--------------|-----------------|---------------|
