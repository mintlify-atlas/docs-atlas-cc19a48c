The `LogPredict.ipynb` notebook implements logistic regression from scratch using gradient ascent with L2 regularization. It trains and evaluates 24 model configurations across 4 feature transformations.

## Purpose

This notebook demonstrates a custom implementation of logistic regression without using scikit-learn's built-in classifier. It explores:

- Gradient ascent optimization for maximum likelihood estimation
- L2 (ridge) regularization to prevent overfitting
- Hyperparameter tuning (learning rate, iterations, regularization strength)
- Performance comparison across feature transformations

## Implementation details

### Core functions

<Accordion title="Sigmoid function">
  ```python
  def sigmoid(z):
      e_term = np.exp(-z)
      return 1/(1+e_term)
  ```
  
  Computes the logistic sigmoid: σ(z) = 1 / (1 + e^(-z))
</Accordion>

<Accordion title="Hypothesis function">
  ```python
  def hypothesis(X, w):
      Xw = np.dot(X, w)
      return sigmoid(Xw)
  ```
  
  Predicts probabilities: h(x) = σ(w^T x)
</Accordion>

<Accordion title="Log-likelihood with L2 regularization">
  ```python
  def log_likelihood(X, y, w, lambda_reg=0):
      hyp = hypothesis(X, w)
      log_lik = np.sum(y * np.log(hyp + 1e-10) + (1 - y) * np.log(1 - hyp + 1e-10))
      
      # Add L2 regularization term (excluding bias term)
      if lambda_reg > 0:
          w_without_bias = w[1:].copy()
          reg_term = -lambda_reg * np.sum(w_without_bias ** 2) / 2
          log_lik += reg_term
      
      return log_lik
  ```
  
  Computes:
  - Log-likelihood: Σ[y log h(x) + (1-y) log(1-h(x))]
  - Regularization penalty: -λ||w||² / 2 (bias term excluded)
</Accordion>

<Accordion title="Gradient ascent">
  ```python
  def gradient_ascent(X, y, w, learning_rate=0.01, num_iterations=1000, lambda_reg=0):
      m = X.shape[0]
      
      for i in range(num_iterations):
          hyp = hypothesis(X, w)
          gradient = np.dot(X.T, (y - hyp)) / m
          
          # Apply L2 regularization to weights (exclude bias)
          if lambda_reg > 0:
              reg_gradient = lambda_reg * w
              reg_gradient[0] = 0  # Don't regularize bias
              gradient -= reg_gradient
          
          w = w + learning_rate * gradient
      
      return w
  ```
  
  Updates weights iteratively: w := w + α∇L(w)
</Accordion>

## Hyperparameter grid

The notebook trains 24 models by testing:

| Hyperparameter | Values tested |
|----------------|---------------|
| Learning rate (α) | 0.01, 0.1 |
| Iterations | 1000, 5000 |
| Regularization (λ) | 0, 0.01, 0.1 |

Across 4 feature transformations:
- Baseline scaled
- Polynomial degree 2
- Polynomial degree 3
- PCA

**Total configurations**: 2 × 2 × 3 × 4 = 48 (notebook tests a subset of 24 representative configs)

## Best model results

The best performing configuration:

- **Feature transformation**: Baseline scaled
- **Learning rate**: 0.1
- **Iterations**: 5000
- **Regularization λ**: 0.01
