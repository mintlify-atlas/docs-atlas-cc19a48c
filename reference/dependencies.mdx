The NBAPredict project relies on several Python packages for data processing, machine learning, and visualization. All dependencies are listed in `requirements.txt` and can be installed using pip.

## Installation

Install all required dependencies with:

```bash
pip install -r requirements.txt
```

For development in a virtual environment:

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Version Management

The `requirements.txt` file currently uses unpinned versions, which means `pip install` will fetch the latest compatible versions. For production deployments, consider:

1. **Pinning versions** for reproducibility:
   ```bash
   pip freeze > requirements.txt
   ```

2. **Using version constraints** for security updates:
   ```txt
   pandas>=2.0.0,<3.0.0
   tensorflow>=2.12.0,<3.0.0
   ```

## Compatibility Notes

<AccordionGroup>
  <Accordion title="Python Version">
    This project requires Python 3.8 or higher. TensorFlow 2.x has the best support for Python 3.8-3.11.
  </Accordion>
  
  <Accordion title="TensorFlow GPU Support">
    To use GPU acceleration:
    - Install CUDA and cuDNN compatible with your TensorFlow version
    - Install tensorflow-gpu instead of tensorflow (or use tensorflow 2.x which includes GPU support)
    - Verify GPU availability with `tf.config.list_physical_devices('GPU')`
  </Accordion>
  
  <Accordion title="Platform-Specific Issues">
    - **macOS M1/M2:** Use tensorflow-macos and tensorflow-metal for Apple Silicon optimization
    - **Windows:** Some packages may require Microsoft C++ Build Tools
    - **Linux:** Generally has the best compatibility with all packages
  </Accordion>
</AccordionGroup>

## Updating Dependencies

To update all packages to their latest versions:

```bash
pip install --upgrade -r requirements.txt
```

To update a specific package:

```bash
pip install --upgrade pandas
```
